# MERN Expense Tracker

A full-stack expense tracker built with the MERN stack.

## Features
- User authentication
- Add income and expenses
- Dashboard analytics
- Expense history
- Secure REST API

## Tech Stack
MongoDB  
Express  
React  
Node.js  
TailwindCSS  
JWT Authentication

## Installation

git clone https://github.com/assumpta15/expense-tracker

cd ExpenseTracker

cd backend
npm install

cd ../frontend
npm install
